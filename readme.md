# TypeScript Test Project

โปรเจกต์นี้เป็นชุดของโจทย์ TypeScript พร้อมการทดสอบด้วย Vitest

## การติดตั้ง

```bash
npm install
```

## การรันเทสต์

รันเทสต์ทั้งหมด:

```bash
npm run test
```

## โครงสร้างโปรเจกต์

- `src/` - ไฟล์ source code และ test files
  - `1_string_check.ts` - ตรวจสอบความถูกต้องของ string
  - `2_array_string_check.ts` - ตรวจสอบ array ของ string
  - `3_autocomplete.ts` - ฟังก์ชัน autocomplete
  - `4_roman_numerals.ts` - แปลงตัวเลขเป็นเลขโรมัน
  - `5_positive_int_sort.ts` - เรียงลำดับตัวเลขบวก
  - `6_tribonacci.ts` - ฟังก์ชัน tribonacci sequence
